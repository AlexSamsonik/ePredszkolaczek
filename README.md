# ePredszkolaczek
A project to calculate the actual number of hours children are in kindergarten.

- Install Virtual Environment (venv):
```sh
python -m venv venv
```

- Activate venv:
```sh
source venv/bin/activate
```

- Install poetry
```sh
pip install poetry
```

- Install dependencies via poetry
```sh
poetry install
```

- Install pre-commit hooks
```sh
pre-commit install
```

- Install playwright
```sh
playwright install
```
